# Activity A02: Samsung Galaxy S25 Ultra Promotional Page

A sleek, responsive, and modern single-page promotional website showcasing the flagship **Samsung Galaxy S25 Ultra**. This project focuses on high-end UI design, featuring custom gradients, interactive navigation, and integrated media.

---

## 📱 Project Overview
This activity was designed to practice advanced web development layouts and responsive design. The goal was to create a visually appealing product landing page that mimics a premium brand experience.

### Key Features:
* **Premium Hero Section:** High-impact visuals with a modern glassmorphism effect.
* **Integrated Media:** Embedded YouTube video demonstration and asset-based video content.
* **Responsive Grid:** A two-column layout that adjusts seamlessly from desktop to mobile.
* **Interactive Navigation:** Custom hover and active states with a vibrant blue theme.
* **Performance Sections:** Detailed highlight of the 1TB storage and 16GB LPDDR5X RAM specifications.

---

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom linear gradients, nested border-radius effects, and smooth transitions.
* **Bootstrap 5:** Utilized for the responsive grid system, spacing utilities, and interactive components.
* **JavaScript:** Logic for dynamic UI updates and navigation link states.
